# Meteo
## Jedndouchá webová stránka s přehledem o počasí
### Vlastnosti
* Aktuální, hodinová a 3-denní předpověď počasí
* Zadávání měst
* aktualizace každých 15 minut
### Problémy (bude opraveno)
* Zadávání měst nedisponuje našeptávačem
* UI není responzivní
